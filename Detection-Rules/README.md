# Detection Rules

This directory contains custom detection rules created during SOC lab exercises.

## Structure

- MITRE/ - MITRE ATT&CK mapped detections
- Queries/ - KQL, EQL, Lucene, Sigma, and Elasticsearch queries
- Screenshots/ - Detection rule screenshots and alert validation

Each detection includes:
- Rule Name
- Objective
- MITRE ATT&CK Technique
- Detection Logic
- Test Method
- Expected Alert
